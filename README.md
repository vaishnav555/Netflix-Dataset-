# Netflix-Dataset-
This project involves cleaning and preparing the Netflix Movies and TV Shows dataset to make it analysis-ready. The dataset contains information about movies and TV shows on Netflix, including titles, directors, cast, country, release year, ratings, genres, and more.

-- Dataset Columns (Cleaned)

show_id – Unique identifier
type – Movie / TV Show
title – Name of the content
director – Director of the content (may have missing values)
cast – Actors
country – Country of production
date_added – Date content was added to Netflix
release_year – Year of release
rating – Content rating (PG, TV-MA, etc.)
duration – Movie duration or number of seasons
listed_in – Genre
Data_Quality_Notes – Analyst notes about data issues

-- Project Objectives

Open and inspect the dataset, identifying missing values and inconsistent columns
Remove duplicates while backing up raw data
Standardize text fields and fix formatting issues
Validate dates and numeric/categorical columns
Maintain a separate Cleaned_Data sheet for analysis-ready data
Document data quality issues in a Data_Quality_Notes column
Export cleaned dataset in Excel (Cleaned_Netflix_Dataset.xlsx) and CSV (cleaned_netflix_dataset.csv)

-- Key Skills Demonstrated

Data cleaning & preprocessing 
Excel-like workflows (Freeze Panes, filters)
Duplicate and missing value handling
Data quality documentation
Analyst thinking & storytelling
